## 🚀 Deploy to GitHub Pages (5 steps)

### Step 1 — Create a GitHub repository
1. Go to [github.com](https://github.com) and sign in
2. Click **+** → **New repository**
3. Name it: `meenu-portfolio` (or any name you prefer)
4. Set visibility to **Public**
5. Click **Create repository**

### Step 2 — Upload the files
1. On your new repo page, click **uploading an existing file**
2. Drag and drop **all three files**:
   - `index.html`
   - `README.md`
   - `.nojekyll`
3. Click **Commit changes**

### Step 3 — Enable GitHub Pages
1. Go to **Settings** → **Pages** (left sidebar)
2. Under **Source**, select **Deploy from a branch**
3. Branch: **main** · Folder: **/ (root)**
4. Click **Save**

### Step 4 — Wait ~60 seconds
GitHub will build and deploy your site automatically.

### Step 5 — Visit your live portfolio
Your portfolio will be live at:
```
https://YOUR-USERNAME.github.io/meenu-portfolio/
```
