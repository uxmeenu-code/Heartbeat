# Meenu M Pillai — Portfolio

## Files
- `index.html` — Portfolio (deploy this to GitHub Pages)
- `image-manager.html` — Open locally to add/replace images
- `image-map.html` — Visual reference showing all image slots

## Deploy to GitHub Pages
1. Upload `index.html` to your repo root
2. Settings → Pages → main branch → / (root)

## How to add/replace images
Open `image-manager.html` in Chrome → drag `index.html` onto it → click any image or placeholder → upload → Download Updated HTML
